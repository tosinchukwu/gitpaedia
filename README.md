# Gitpaedia

Learn Git interactively.

## Setup

1. Copy `.env.example` to `.env` and fill in your Supabase and Privy keys.
2. Run `npm install`
3. Run `npm run dev`

## Database

Create a table `user_progress` with columns: `id`, `privy_user_id`, `level_completed`, `badges`, `total_xp`, `streak`, `updated_at`.

## Deployment

Push to GitHub and deploy on Vercel.
